Kumpulan Command untuk Github:
1. touch (ini untuk membuat file baru di repository)
contoh : 
touch index.js (ini akan menambahkan file index.js ke repository)

2. git init (ini akan membuat repository baru di folder tertentu)

3. git config --global user.name 'ramadhaniabidin' (ini untuk menghubungkan dengan akun github)
4. git config -- global user.email 'ramadhaniabidin12@gmail.com' (ini juga sama dengan nomor 3)
5. git add namafile (ini untuk menambahkan file ke dalam repository)
6. git status 
7. git commit -m 'message commit'
8. git clone (untuk clone dari repositroy di Github online)
9. git push (upload ke Github online)
10. git pull (download dari github online)
